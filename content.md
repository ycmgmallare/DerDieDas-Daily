This is the **starter content** for your email automation. The automation sends
whatever lives in this file — the body, the `subject`, and the `title` above.

## How to change it

- Edit this file by hand, **or**
- Generate it with another skill (e.g. run `/write-post`) and have it overwrite this file.

Then send from the `/admin` console, or wait for the daily schedule. Point at a
different file with `CONTENT_FILE` in `.env`.
